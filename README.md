# ❤️ Heart Disease Prediction

A Machine Learning project that predicts whether a person is at risk of heart disease based on various medical attributes.

## 📌 Project Overview

This project uses the **Logistic Regression** algorithm to build a classification model that predicts the likelihood of heart disease. The model is trained on a labeled dataset and can classify whether a patient is likely to have heart disease based on the provided health information.

## 🚀 Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Model training using Logistic Regression
- Model evaluation and accuracy testing
- Predictive system for new patient data

## 🛠️ Technologies Used

- **Language:** Python
- **Environment:** Google Colab
- **Libraries:**
  - NumPy
  - Pandas
  - Scikit-learn

## 🤖 Machine Learning Details

- **Problem Type:** Supervised Learning
- **Task:** Binary Classification
- **Algorithm:** Logistic Regression

## ▶️ How to Run

1. Clone this repository.
2. Open the notebook in **Google Colab** or **Jupyter Notebook**.
3. Install the required Python libraries.
4. Run all notebook cells.
5. Enter patient data to test the prediction model.
